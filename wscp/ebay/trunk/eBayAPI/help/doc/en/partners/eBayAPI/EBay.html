EBay Item Information PowerToy

This documentation contains instruction on how to:

  • Configure the EBay Item Information Control

  • Use the EBay Item Information Control

The Ebay control makes use of the EBay Developers Program. To get familiar with ebay developers program, please click here. Also please take a look here for services/fees for using API. EBay Item Information control uses the EBAY APIs, documented here. The function used by this control is "GetItem" which belongs to the API mentioned above. Before you can use the EBay API, you need to register with EBay's developer program and obtain set of keys required for this control e.g. userid, password, devID, appID, certID. You need to configure the EBay Control with these data items key, as described in the section on configuring the control.
