UPS Shipment Tracking PowerToy

This documentation contains instruction on how to:

  • Configure the UPS Shipment Tracking Control

  • Use the UPS Shipment Tracking Control

The UPS Shipment Tracking Control uses the UPS Web APIs, documented here. Before you can use the UPS Web APIs, you need to register with UPS and obtain an access key and a username/password. You need to configure the UPS Control with these data items key, as described in the section on configuring the control.
