@jc:sforce-query-options Annotation

Settable query options on the sForce servicE

Syntax

@jc:sforce-query-options

batch-size="<data-source-type>"

Attributes

batch-size

Maximum number of records to return on the query method

Default: none.

Remarks

The following rules apply to this annotation's use:

  • A single @jc:sforce-query-options annotation may appear within a single Javadoc comment block.
